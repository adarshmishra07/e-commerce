# Ecommerce Website

## A MERN Stack Build
  
### Server Side 
  
  - NodeJs
  - MongoDB
  - JWT
  - Express
  
  
  ### Client Side
  
  - ReactJs
  - Braintree payment gateway
  
  
### Features 

- Admin DashBoard
  - Manage products
  - Manage Orders
  - Manage Categories
  - Create Products
  - Create Categories
  
- User Side 
  - Add item to cart 
  - Proceed to checkout
  - Payment gateway
